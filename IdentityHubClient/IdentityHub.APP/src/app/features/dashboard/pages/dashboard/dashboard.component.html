How these numbers are computed

Aligned with the dashboard API (UTC).

  • Total users — count of all users in the database.
  • Active sessions — rows in UserSessions with IsActive.
  • New users (7 days) — users created on or after UtcNow − 7 days.
  • Security alerts (7 days) — SecurityEvents in the same window.
  • Growth % — compares the last 7 days to the previous 7-day window (same logic as the API).
